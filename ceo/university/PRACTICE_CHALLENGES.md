# 🎯 Praxis-Lessons — Live Coding Challenges
*Erstellt: 2026-04-09 22:43 UTC*

---

## 📋 Konzept

Praxis-Lessons sind echte Aufgaben statt Theorie.
Agents müssen Probleme lösen, Code schreiben oder Systeme debuggen.

---

## 🎮 Challenge Types

| Type | Beschreibung | Beispiele |
|------|-------------|-----------|
| **Code** | Script schreiben | Python Script, Bash, API |
| **Debug** | Bug finden & fixen | Error-Analyse, Fix |
| **Build** | System aufsetzen | Docker, Config, Pipeline |
| **Review** | Code prüfen | Security Audit, QC |
| **Optimize** | Performance verbessern | Indexing, Caching |

---

```markdown
# 🎯 Challenge: [Titel]

- [ ] Kriterium 3

[Wenn Agent nicht weiterkommt]

[Was soll abgegeben werden?]
```

## 🎯 Beispiel-Challenges

### Challenge 1: Security Audit Script
```
Schwierigkeit: 🔴 Schwer
Agent: Security Officer
XP: 100

Erstelle ein Security Audit Script das:
- Alle Config-Files prüft
- Auf sensitive Data prüft
- Fehlende Permissions meldet
- Report generiert
```

### Challenge 2: Heartbeat Monitor
```
Schwierigkeit: 🟡 Mittel
Agent: Builder
XP: 75

Erstelle ein Heartbeat Monitoring Dashboard:
- Alle Agent Heartbeats visualisiert
- Timeout-Alerts
- History Graph
```

### Challenge 3: QC Validation Framework
```
Schwierigkeit: 🟡 Mittel
Agent: QC Officer
XP: 75

Erstelle ein Validation Framework:
- Checkliste pro Task-Type
- Auto-Validation
- Report-Generation
```

## 🔄 Peer-Review System

Nach einer Challenge:
1. Agent A löst Challenge → submit
2. Agent B reviewed Lösung → Feedback
3. CEO validiert → XP vergeben
