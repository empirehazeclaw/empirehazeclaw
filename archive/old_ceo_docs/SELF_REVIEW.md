## 🎯 SKILL VETTING WORKFLOW (2026-04-10)

### was ich gelernt habe:
- ClawHub hat eingebaute Security-Flags ("SUSPICIOUS")
- `clawhub inspect --files` zeigt alle Dateien vor Installation
- `clawhub inspect --file <path>` liest einzelne Dateien
- install.mjs Scripts sind IMMER riskant — nie blind installieren
- Zero dependencies ist gut, aber nicht genug

### Neuer Standard:
Für JEDEN ClawHub Skill:
1. Suchen mit npx clawhub search
2. Inspect mit --files
3. Security-Analyse aller Files
4. Niemals direkt installieren
