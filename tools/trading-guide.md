## 📈 Trading Strategien

### 1. Trend Following

```
1. Trend identifizieren (EMA 50/200)
2. Pullback abwarten
3. Einstieg beim Support
4. Stop-Loss unter Support
5. Take Profit beim nächsten Resistance
```

### 2. Swing Trading

```
1. Range identifizieren
2. Support kaufen
3. Resistance verkaufen
4. Risiko:Belohnung min 1:2
```

### 3. DCA (Dollar Cost Averaging)

```
1. Gleiche Beträge kaufen
2. Regelmäßig (Woche/Monat)
3. Langfristig halten
```

## 🛡️ Risiko-Management

### Die goldenen Regeln

| Regel | Beschreibung |
|-------|--------------|
| **1. Nie mehr 2% pro Trade** | Kapital erhalten |
| **2. Risk/Reward min 1:2** | Gewinn > Verlust |
| **3. Stop-Loss IMMER** | Verlust begrenzen |
| **4. Diversifizieren** | Nicht alles in einen Trade |
| **5. Emotionen raus** | Plan befolgen |

### Positionsgröße

```
Positionsgröße = (Kapital × Risiko%) / (Entry - Stop-Loss)
```

Beispiel:
- Kapital: €10.000
- Risiko: 2% = €200
- Entry: €100
- Stop-Loss: €90 (10% Verlust)

€200 / €10 = €20 investieren = 0.2 Coins
