## 📧 RESERVIERUNGS-EMAILS

### Prompt 1: Reservierungsanfrage beantworten
```
Schreibe eine professionelle Email-Bestätigung für eine Restaurant-Reservierung.

Details:
- Name: [Gast Name]
- Datum: [Datum]
- Uhrzeit: [Uhrzeit]
- Personenzahl: [Anzahl]
- Telefon: [Telefon]

Ton: Freundlich, professionell, persönlich
Enthält: Bestätigung aller Details + Rückfrage ob Special Requests
Länge: 150-200 Wörter
```

### Prompt 2: Reservierungsablehnung (volles Haus)
```
Schreibe eine höfliche Email, die eine Reservierungsanfrage ablehnt, weil das Restaurant ausgebucht ist.

Ton: Entschuldigend, kulant, biete Alternative an
Enthält: 
- Ehrliche Begründung
- Alternative Daten vorschlagen
- Auf Warteliste anbieten
- Lokale Empfehlungen

Länge: 120-150 Wörter
```

### Prompt 3: Reservierungsänderung bestätigen
```
Schreibe eine Bestätigungs-Email für eine geänderte Reservierung.

Original:
- Datum: [Datum alt]
- Uhrzeit: [Zeit alt]
- Personen: [Anzahl alt]

Neu:
- Datum: [Datum neu]
- Uhrzeit: [Zeit neu]
- Personen: [Anzahl neu]

Ton: Klar, bestätigend, nochmal alle wichtigen Infos
Länge: 100-130 Wörter
```

### Prompt 4: Reservierungs-Erinnerung (24h vorher)
```
Schreibe eine freundliche Erinnerungs-Email 24 Stunden vor der Reservierung.

Details:
- Restaurant: [Name]
- Adresse: [Adresse]
- Uhrzeit: [Uhrzeit]
- Personenzahl: [Anzahl]
- Kleidung: [Dresscode falls vorhanden]
- Parken: [Parkinfos]
- Special: [z.B. Geburtstag]

Ton: Warm, einladend, nützliche Infos
Länge: 80-100 Wörter
```

### Prompt 5: No-Show Follow-up Email
```
Schreibe eine professionelle Email nach einem No-Show ohne Absage.

Ton: Verständnisvoll, nicht beschuldigend
Enthält:
- Freude über vergangene Besuche
- Verständnis für unvorhergesehene Ereignisse
- Einladung neu zu reservieren
- Vielleicht kleiner Anreiz (Aperitif, etc.)

Länge: 100-130 Wörter
```
