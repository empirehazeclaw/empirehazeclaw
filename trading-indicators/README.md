# Trading Indicators Collection

Professionelle Trading Indicators für technische Analyse.

## Enthaltene Indikatoren

### 1. Moving Averages
- SMA (Simple Moving Average)
- EMA (Exponential Moving Average)
- WMA (Weighted Moving Average)

### 2. Momentum
- RSI (Relative Strength Index)
- Stochastic Oscillator
- MACD

### 3. Volatilität
- Bollinger Bands
- ATR (Average True Range)
- Standard Deviation

### 4. Trend
- ADX (Average Directional Index)
- Ichimoku Cloud
- Parabolic SAR

## Verwendung

```python
from indicators import RSI, MACD, BollingerBands

# RSI Beispiel
rsi = RSI(prices, period=14)
if rsi < 30:  # Oversold
    print("KAUFEN")
elif rsi > 70:  # Overbought
    print("VERKAUFEN")
```
