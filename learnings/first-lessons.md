# Lessons Learned

## 2026-03-14

### What worked:
- Tavily: Excellent for searching and finding emails
- Resend API: Easy to integrate
- Night Shift Automation: Running smoothly

### What didn't:
- Resend Free: Only 5 emails/day limit
- Some startup emails were invalid

### Insights:
- Crawl4AI needs browser installation (more complex)
- Crunchbase is paid, LinkedIn better for free
- Email deliverability matters

### Action items:
- Get Resend Pro for unlimited emails
- Verify emails before sending
- Build own email list organically
