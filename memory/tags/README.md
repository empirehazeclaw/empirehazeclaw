# Tag Hierarchy

## Main Tags

```
#idea               - Neue Ideen
  #idea-marketing   - Marketing Ideen
  #idea-product     - Produkt Ideen
  #idea-content     - Content Ideen

#learning          - Gelernte Lektionen
  #learning-tech    - Technisches Know-How
  #learning-business - Business Insights

#project           - Projektbezogen
  #project-outreach - Outreach Kampagnen
  #project-content - Content Projekte

#review            - Review/Feedback
#decision         - Wichtige Entscheidungen
```

## How to Use

When creating a note:
```bash
python3 memory/scripts/quick_capture.py "Neue Marketing Idee" --tags idea,marketing
```

This creates a note with both `#idea` and `#idea-marketing` tags.

## Benefits

- More specific than flat tags
- Easier to find related notes
- Hierarchy shows relationships
