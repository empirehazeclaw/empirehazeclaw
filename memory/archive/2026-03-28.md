### Read Inbox via Gmail API
```bash
TOKEN=$(cat ~/.config/gogcli/token.env | grep access_token | cut -d= -f2)
curl -s "https://gmail.googleapis.com/gmail/v1/users/me/messages?maxResults=20" \
  -H "Authorization: Bearer $TOKEN"
```
