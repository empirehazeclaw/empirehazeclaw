## 🧠 Learning Methods

### 1. Pattern-Based Learning
```
AFTER every success:
  → Extract pattern with pattern_extractor.py
  → Store in skills/_library/
  → Update KG with new entity
```

### 2. Error-Based Learning
```
AFTER every error:
  → Log error type in session_analyzer
  → Create/update troubleshooting skill
  → Fix immediately or document for later
```

### 3. Skill Performance Learning
```
EVERY evening:
  → Run skill_tracker.py --report
  → Identify low-scoring skills
  → Focus tomorrow's learning on weakest skill
```

### 4. Research Integration
```
DAILY (14:00 UTC):
  → Innovation Research
  → KG Update
  → Best patterns → Skill Library
```

### 5. Reflection Loop
```
BEFORE every task:
  → Similar task before? (Check KG)
  → What worked last time?
  → Apply bewährten Weg (Rule 4b)
```
