# ✍️ Content Creator Skill

Content Erstellung für EmpireHazeClaw's Marketing.

## Nutzung

```bash
# Text humanizen (AI-Detection umgehen)
python3 /home/clawbot/.openclaw/workspace/scripts/humanize_content.py --input text.txt

# Semantic Search für Content Ideen
python3 /home/clawbot/.openclaw/workspace/scripts/semantic_search.py --query "KI Agent Marketing"
```

## Kern-Scripts

| Script | Zweck |
|--------|-------|
| `humanize_content.py` | Text naturalisieren für Social Media |
| `semantic_search.py` | Vektor-Suche für Content Ideen |
