## Purpose

Self-improvement and continuous learning for Sir HazeClaw.

## Tools

### Test Framework
```bash
python3 scripts/test_framework.py [--run <test>] [--list]
```
Runs automated tests on important scripts.

### Self-Evaluation
```bash
python3 scripts/self_eval.py [--report]
```
Evaluates current state vs defined goals.

### Quality Metrics
```bash
python3 scripts/quality_metrics.py [--days 7] [--save]
```
Tracks quality metrics over time.

### Deep Reflection
```bash
python3 scripts/deep_reflection.py [--output all]
```
Deep self-reflection for continuous improvement.

### Auto Documentation
```bash
python3 scripts/auto_doc.py [--update-readme]
```
Auto-generates documentation from scripts.

## Goals Tracked

| Goal | Target | Weight |
|------|--------|--------|
| Quality über Quantität | Backup Ratio < 0.3, Test Coverage > 80% | High |
| Produktivität | > 5 Commits/day | Medium |
| Wissen & Lernen | > 100 KG entities | Medium |
| Self-Management | Daily updates, Weekly reflection | High |

## Usage

1. **Before large task:** Run self_eval to know your current state
2. **After task:** Run test_framework to verify quality
3. **Daily:** Run quality_metrics to track progress
4. **Weekly:** Run deep_reflection for insights
