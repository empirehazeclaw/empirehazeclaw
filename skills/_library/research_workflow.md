# 🔍 Research Workflow Skill
**Created:** 2026-04-11
**Category:** research
**Times Used:** 0

## When to Use
When finding information, evaluating tools, or learning new patterns.

## Workflow
```
1. DEFINE - Was will ich wissen?
2. SEARCH - Web-Suche, Docs, Code
3. EVALUATE - Quellen prüfen
4. SYNTHESIZE - Zusammenfassung schreiben
5. APPLY - Erkenntnisse anwenden
```

## Sources Priority
1. Offizielle Docs (OpenClaw, npm packages)
2. Web Search (Brave Search)
3. Source Code Inspection
4. Community (Discord, Reddit)

## Anti-Patterns (Vermeiden)
- ❌ Nicht ewig recherchieren ohne zu handeln
- ❌ Nicht 10 Quellen lesen wenn 2 reichen
- ❌ Nicht dokumentieren ohne zu implementieren

- [ ] Klare Antwort gefunden
