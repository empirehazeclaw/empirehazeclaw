## Kurzbeschreibung

Das Memory System ist ein **produktionsreifes Gedächtnis-System** für OpenClaw AI-Agenten. Es löst das Problem von veralteten, unorganisierten Textdateien durch ein dreistufiges System: PARA-Wissensgraph mit automatischem Fakten-Extraction, tägliche Notizen als chronologische Timeline, und stilles Wissen über den Benutzer.

## Hauptfunktionen

- **Automatisches Lernen:** Extrahiert Fakten automatisch aus jeder Nachricht
- **Memory Decay:** Alte Fakten verblassen (9% pro Woche)
- **PARA-Organisation:** Projects, Areas, Resources, Archives
- **Prioritätsverwaltung:** CRITICAL → HIGH → MEDIUM → LOW
- **Zugriffstracking:** Weiß, was wirklich genutzt wird

## Für wen?

AI-Entwickler, die wollen dass ihr Agent sich an Gespräche erinnert, lernt, und Wissen priorisiert. Perfekt für Chatbots, Assistenten, und autonome Agenten.
